# doutor_MM1_queue
simulating the waiting line in a doutor coffee shop

ドトール本郷店の待ち行列をシミュレーションしている。

待ち行列のM/M/1モデルに関しては、https://en.wikipedia.org/wiki/M/M/1_queue を参照。

ただし、本郷店では、列が3人以上になったときに、店員が一人増えて対応し、列がなくなるまでM/M/2モデルに変化する。
